# Curso de Arduino Uno - Prácticas

Este repositorio contiene las prácticas del curso de Arduino Uno. Cada carpeta corresponde a una clase práctica con explicaciones teóricas, código fuente y ejercicios.

## Estructura del repositorio

### Prácticas disponibles

- **[Clase práctica 01](https://github.com/gaaguirrer/ArduinoUNO/tree/main/Clase%20pr%C3%A1ctica%2001)** - Parpadeo de un LED (Blink)
  - `Práctica #01 Parpadeo de un LED.md`: Encendido y apagado de un LED en el pin 12 con delays de 300 ms. Incluye preguntas teóricas y ejercicios prácticos.

- **[Clase práctica 02](https://github.com/gaaguirrer/ArduinoUNO/tree/main/Clase%20pr%C3%A1ctica%2002)** - Variables y secuencias con múltiples LEDs
  - `Práctica #02 - P1 _ prendido y apagado de leds ordenados - Declaración de Variables.md`: Secuencia individual (rojo, amarillo, verde) con variables.
  - `Práctica #02 - P2 _ Encendido y apagado de leds - Declaración de Variables.md`: Encendido acumulativo y apagado decremental (efecto de llenado y vaciado).
  - `Práctica #02 - P3 _ Semáforo - Declaración de Variables.md`: Simulación de semáforo con tiempos variables.
  - `Práctica #02 - P4 _ led RGB- Declaración de Variables.md`: Control de LED RGB para mostrar siete colores.

- **[Clase práctica 03](https://github.com/gaaguirrer/ArduinoUNO/tree/main/Clase%20pr%C3%A1ctica%2003)** - Entradas analógicas, pulsadores y estructuras de control
  - `Práctica #03 - P1 _ Control de Distribución de voltaje - Declaración de Variables.md`: Potenciómetro y PWM con brillo complementario entre dos LEDs.
  - `Práctica #03 - P2 _ LED con Pulsador - IF.md`: Secuencia de LEDs solo al presionar un pulsador (uso de `if`).
  - `Práctica #03 - P3 LED con Pulsador - IF.md`: Interruptor toggle con variable `bool` y detección de flanco.
  - `Práctica #03 - P4 _ LED con Pulsador - While.md`: Toggle usando bucles `while` bloqueantes.

## Requisitos generales

- Arduino Uno (o compatible) o acceso a Tinkercad (los enlaces están incluidos en cada práctica).
- Componentes: LEDs, resistencias de 220 Ω y 10 kΩ, potenciómetro, pulsadores, LED RGB, cables y protoboard.

## Uso

Cada archivo de práctica contiene:
- Explicación del código línea por línea.
- Preguntas teóricas.
- Ejercicios prácticos para modificar el código y observar cambios.
- Enlace al simulador en Tinkercad (cuando está disponible).

Se recomienda seguir el orden numérico de las carpetas y dentro de cada una el orden de los archivos (P1, P2, …).
